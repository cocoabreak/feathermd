## Reviewed behavior

The following release behavior was reviewed during development:

- Native and ZIP document browsing, trusted-root enforcement, and bounded file watching
- Session restoration across actual WebView reloads
- Large-document safe mode and full-render opt-in
- External image privacy policies and sanitization boundaries
- Search, keyboard navigation, accessibility, and pane resizing
- HTML, SVG, and PNG export through a native save dialog without general WebView file-write permission
- Help/About content, licenses, repository links, and English/Japanese README presentation
- Windows shell integration, single-instance behavior, and update notification

## Known release notes

- Windows binaries are not code-signed and may trigger Microsoft Defender SmartScreen. Users should obtain binaries only from the official Releases page and verify the source before running them.
- The low-severity npm advisories are inherited through a development-only SvelteKit dependency path. FeatherMD uses the static adapter and does not ship a SvelteKit server; moderate-or-higher findings remain release-blocking.
- Production frontend builds may report existing large-chunk and dynamic-import optimization warnings. These warnings are non-blocking for v0.2.2.
