# 要求定義: テーマ・スタイル設定 (theme-settings)

## 背景・動機
長時間の閲覧において、環境（昼夜など）に合わせたテーマ（ライト/ダーク）の切り替えが必須である。
また、Markdown内のコードブロックもユーザーの好みのハイライトテーマで表示したい。

## ユーザーストーリー
### US-001: アプリ全体のテーマ選択
```
As a ユーザー
I want to アプリのテーマ（ライト、ダーク、システム連携）を選択できる
So that 自分の好みやOSの設定に合わせた見やすい画面で閲覧できる
```
- [x] 設定画面で「ライト」「ダーク」「システム」を選択できる
- [x] システム連携の場合、OSのテーマ変更に即座に追従する
- [x] テーマは次回起動時にも維持される（`settings-store`で永続化）
- [x] ネイティブメニューもアプリのテーマ選択に追従する

### US-002: コードブロックのテーマ・行番号設定
```
As a ユーザー
I want to コードブロックのシンタックスハイライトのテーマと、行番号の表示/非表示を設定できる
So that コードがより読みやすくなる
```
- [x] 設定画面でコードブロック用テーマ（例: GitHub Light, One Dark 等数種類）を選択できる
- [x] 行番号のON/OFFを切り替えられる
