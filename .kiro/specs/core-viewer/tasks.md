- **依存**: T-004
- **概要**: `read_directory` Tauriコマンドを実装する。ディレクトリを再帰的に走査し、ファイルツリー構造を返す
- **返却型**:
  ```rust
  struct FileEntry {
    name: String,
    path: String,
    is_dir: bool,
    children: Option<Vec<FileEntry>>,
    is_hidden: bool,  // '.' 始まりかどうか
  }
  ```
- **完了条件**:
  - [x] ディレクトリパスを受け取り `FileEntry` のツリーを返す
  - [x] `is_hidden` フラグが正しく設定される

- **対応US**: US-002

### T-008: ファイル監視コマンドとイベント ✅

- **対応US**: US-004

### T-009: ブラウザ起動コマンド ✅
