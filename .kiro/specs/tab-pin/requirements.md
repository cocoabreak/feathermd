# 要求定義: タブのピン留め (tab-pin)

## 背景・動機

タブを多数開いていると、意図せず`Ctrl+W`やクローズボタンで重要なタブを閉じてしまうことがある。よく参照するファイルを「ピン留め」して閉じられないようにしたい。

## スコープ

- **対象**: タブのピン留め・解除、ピン留めタブのクローズ防止（`Ctrl+W`・クローズボタン共通）
- **対象外**: ピン留めタブの並び替え・先頭固定（`.kiro/backlog.md`のタブ並び替えと合わせて別途検討）、セッション復元（現状タブ状態は永続化されないため、ピン留め状態もアプリ再起動で消える）

---

## ユーザーストーリー

### US-001: タブのピン留め・解除

```
As a ユーザー
I want to タブをピン留め・解除する
So that 重要なファイルのタブを誤って閉じないようにできる
```

**受け入れ条件**

- [x] 各タブにピン留めアイコンボタンがあり、クリックでピン留め状態がトグルされる
- [x] ピン留め状態はアイコンの見た目（Pin/PinOff）で判別できる

---

### US-002: ピン留めタブのクローズ防止

```
As a ユーザー
I want to ピン留めしたタブが誤操作で閉じられない
So that 安心して他のタブを閉じたり操作したりできる
```

**受け入れ条件**

- [x] ピン留めされたタブは、クローズボタン（✕）クリックで閉じられない
- [x] ピン留めされたタブは、`Ctrl+W`（`tab.close`コマンド）でも閉じられない
- [x] ピン留めされていないタブの動作（クローズボタン・`Ctrl+W`とも）は従来通り

---

## 非機能要求

なし（既存のタブ操作と同程度の軽量な状態管理で十分）

## 未決定事項（設計フェーズで決定）

- [x] ピン留めアイコンの配置（クローズボタンとの位置関係）
- [x] クローズボタンをピン留め中も表示するか、非表示にするか
