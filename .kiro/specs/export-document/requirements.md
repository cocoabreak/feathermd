# 要求定義: エクスポート機能 (export-document)

## 背景・動機

Markdownプレビューの結果（美しい状態）を他者に共有したり、印刷したりするために、PDFやHTML出力が必要。

## ユーザーストーリー

### US-001: PDF/HTML 出力

```
As a ユーザー
I want to 表示中のMarkdownをPDFまたはHTMLとしてエクスポートする
So that レンダリングされたドキュメントを他者と共有できる
```

- [x] ファイルメニューまたはツールバーから「エクスポート（PDF / HTML）」を選択できる（PDF: ファイルメニュー・コンテキストメニュー・コマンドパレット / HTML: コンテキストメニュー）。`Ctrl+P`はquick-open-command-palette specでクイックオープンへ再割当した。
- [x] OSのファイル保存ダイアログが開き、保存先を指定できる（PDFはOSの印刷ダイアログ経由で「PDFとして保存」）
- [x] 保存されたファイルは、アプリ上で見えているデザイン（CSS）をある程度保持している
- [x] 保存先はRustのネイティブダイアログ内だけで扱い、WebViewへパスや継続的な書込権限を返さない
- [x] HTML/SVG/PNGの形式・拡張子・64 MiB上限をRust側で検証する
- [x] PNGはCanvas生成前に一辺16,384px・総25,000,000画素を上限として検証する
