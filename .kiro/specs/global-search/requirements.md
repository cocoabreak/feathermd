# 要求定義: グローバル検索 (global-search)

## 背景・動機
ディレクトリ内に多数のMarkdownファイルがある場合、特定のキーワードが含まれるファイルを探し出す機能が必要。

## ユーザーストーリー
### US-001: ディレクトリ内検索
```
As a ユーザー
I want to エクスプローラーで開いているディレクトリ内の全Markdownからテキストを検索する
So that 探している情報がどのファイルにあるか見つけることができる
```
- [x] 検索パネル（またはダイアログ）を開き、キーワードを入力できる
- [x] 検索結果として、該当するファイル名と該当箇所の一部（スニペット）がリスト表示される
- [x] 結果をクリックすると、そのファイルがタブで開き、該当箇所へジャンプ（またはハイライト）する\n
