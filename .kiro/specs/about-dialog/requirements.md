# 要求定義: Aboutダイアログ (about-dialog)

## 背景・動機

アプリのバージョン、著作権、免責事項、内蔵プラグインと主要コンポーネントの情報を、設定項目と混在させず確認できる場所を提供する。

## スコープ

- **対象**: ヘルプメニュー、Aboutダイアログ、プラグイン／コンポーネント情報、ライセンス情報
- **対象外**: 設定画面の再編、プラグインの有効・無効操作、全依存パッケージのライセンス全文生成

## ユーザーストーリー

### US-001: Aboutを開く

- [x] ネイティブメニューに「ヘルプ / Help」を追加する
- [x] ヘルプメニューの「FeatherMDについて / About FeatherMD」からAboutダイアログを開ける
- [x] ダイアログをEsc、閉じるボタン、外側クリックで閉じられる
- [x] ダイアログ内でフォーカスを循環し、閉じた後に元の要素へフォーカスを戻す

### US-002: 基本情報を確認する

- [x] ロゴ、アプリ名、正確なアプリバージョンを表示する
- [x] `Copyright © 2026 Hirofumi Akiyama` とMIT Licenseを表示する
- [x] FeatherMDが他の「Feather」製品・プロジェクトと提携・承認関係にないことを表示する
- [x] 公式GitHubリポジトリを既定ブラウザーで開ける

### US-003: プラグインとコンポーネントを確認する

- [x] 「プラグインとコンポーネント」を折りたたみ表示する
- [x] 内蔵プラグインごとに名前、プラグインバージョン、有効・無効状態を表示する
- [x] 外部レンダリングエンジンを使うプラグインはエンジン名と正確なビルド時バージョンを表示する
- [x] 主要コンポーネントの名前と正確なビルド時バージョンを表示する
- [x] 依存バージョンは `package-lock.json` からビルド時に取得し、表示用の手書きコピーを持たない

### US-004: ライセンスを確認する

- [x] 「ライセンス」を折りたたみ表示する
- [x] FeatherMDのMITライセンス表記を表示する
- [x] 表示対象の主要コンポーネントについてライセンス識別子を表示する

## 未決定事項（設計フェーズで決定）

- [x] 情報の配置: 設定画面ではなく独立Aboutダイアログ
- [x] 詳細情報: 「プラグインとコンポーネント」「ライセンス」の2つの折りたたみ
- [x] バージョン情報源: アプリは`package.json`、依存は`package-lock.json`をビルド時に読む
