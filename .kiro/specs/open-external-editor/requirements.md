# 要求定義: 外部エディタ連携 (open-external-editor)

## 背景・動機
本アプリはビューワーであるため、誤記を見つけたり編集したくなった際に、スムーズに使い慣れたエディタ（VS Codeなど）に遷移したい。

## ユーザーストーリー
### US-001: 外部エディタで開く
```
As a ユーザー
I want to ボタン一つで現在開いているファイルを外部のデフォルトエディタで開く
So that 閲覧中に見つけた修正箇所をすぐに編集できる
```
- [x] ツールバーやタブのコンテキストメニューに「外部エディタで開く」アクションが存在する
- [x] OSで `.md` に関連付けられているデフォルトアプリケーションで該当ファイルが開く\n
